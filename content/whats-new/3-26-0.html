Tabular Editor 3.26.0

Release blog: March 2026 (v. 3.26.0, release notes)

  • AI Assistant (Public Preview): A built-in assistant for common semantic model development tasks -- explore your model, write DAX queries, generate C# scripts, run BPA analysis and more. Bring your own API key and privacy guards. Read the announcement.
  • Run script with preview: A new option for C# scripts that snapshots your model before execution and shows a side-by-side diff of all changes, letting you review and accept or revert them.
  • New built-in BPA rule: Detects User-Defined Functions without compound names (separators), preventing future naming conflicts with built-in DAX functions.
  • Expanded C# scripting accessors: Complete singular and plural Selected.* accessor support for all object types, including roles, KPIs, calendars, calculation items, functions and more.
  • Workspace database improvements: Tabular Editor now detects changes to incremental refresh Source Expression or Polling Expression and prompts before overwriting workspace partitions.
  • TOM Explorer table groups toggle: New toolbar button to show or hide table groups directly from the TOM Explorer.
  • New DAX function support: TABLEOF, NAMEOF (table references), DATESINPERIOD EndBehavior parameter.
  • Various bugfixes including Calendar Editor, DAX Debugger Call Tree, M Editor comment highlighting and C# script editor autocomplete. See release notes for details.
