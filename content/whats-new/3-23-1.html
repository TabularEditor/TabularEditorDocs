Tabular Editor 3.23.1

Release blog: August 2025 (v. 3.23.1, release notes)

  • Table Import Wizard now supports Fabric SQL Databases and Fabric Mirrored Databases as data sources, for both Import and Direct Lake mode models (in line with the Power BI Desktop July 2025 update).
  • Native ARM64 support.
  • Day 0 support for upcoming Power BI features.
  • Messages view now copies the selected cell by default (Ctrl+Shift+C copies the whole row). New right-click context menu for Copy Cell/Copy Row.
  • New word wrap option in toolbar, menu (Edit > Word Wrap), and preferences (Text Editors > General). Line wrap indicators can also be enabled.
  • Apply or Apply selection actions on DAX queries now support DEFINE COLUMN and DEFINE TABLE statements, creating or updating calculated columns/tables as needed.
  • Various bugfixes, see release notes for more details.
