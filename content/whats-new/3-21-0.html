Tabular Editor 3.19.0

Release blog: December 2024 (v. 3.19.0, release notes)

  • More Diagram view improvements:
    • Icons visually indicate column data type
    • A chevron toggle to quickly toggle between displaying all columns, key columns only, or no columns at all
    • Double-arrows when a relationship uses bi-directional cross filtering
  • DAX Optimizer users can now also view the results for RLS and Calculation Item expressions through our DAX Optimizer Integration
  • Script individual objects into TMDL Scripts through right-clicking on the object in the TOM Explorer
  • New API property for the Model object: MetadataSource to help iterate through the folder a model is stored in.
  • And more...
