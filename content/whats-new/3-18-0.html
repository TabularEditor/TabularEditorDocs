Tabular Editor 3.18.0

Release blog: October 2024 (v. 3.18.0, release notes)

  • DAX Optimizer integration is GA and includes 4 free runs per day for Tabular Editor 3 Enterprise Edition customers
  • One-click DAX improvements and refactorings with the new code actions feature
  • Explicit authentication against XMLA endpoints, using Microsoft Entra MFA
  • Customize how Auto-exists work with the new model property Value Filter behavior
  • Set your own number of recent items in the preferences - default is 10
  • Fixed several bugs including ensuring DAX Debugger support for FormatString measures in DAX Queries
  • And more...
