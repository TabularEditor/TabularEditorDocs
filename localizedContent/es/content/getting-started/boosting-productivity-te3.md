Esta sección contiene varios artículos que te permitirán familiarizarte rápidamente con Tabular Editor 3 y sus principales características, con especial énfasis en cómo hacer el trabajo de la forma más productiva posible.

A continuación tienes un resumen de los artículos y lo que aprenderás en cada uno:

- @importing-tables-data-modeling: Este artículo explica cómo importar tablas, actualizar los esquemas de las tablas y visualizar y editar relaciones.
- @refresh-preview-query: Este artículo explica cómo programar operaciones de actualización, cómo previsualizar datos de tablas, cómo ver datos resumidos mediante Pivot Grid, cómo crear y ejecutar consultas DAX y cómo usar la versión integrada de [Analizador VertiPaq](https://www.sqlbi.com/tools/vertipaq-analyzer/) en Tabular Editor 3.
- @creating-and-testing-dax: Este artículo presenta el potente editor de DAX de Tabular Editor 3 y muestra cómo agregar y editar rápidamente cálculos DAX en tu modelo, como medidas, columnas calculadas y tablas calculadas.
- @dax-script-introduction: Este artículo muestra cómo usar la funcionalidad de scripting de DAX para escribir, mantener y probar lógica de negocio compleja en varias medidas, combinando su definición en un único script DAX.
- @bpa: Este artículo ofrece más detalles sobre el Best Practice Analyzer y explica cómo configurar reglas de mejores prácticas personalizables para mejorar la calidad del código de tus modelos.
- @cs-scripts-and-macros: Este artículo presenta los conceptos de scripts C# en Tabular Editor 3, el Grabador de scripts y cómo guardar scripts como macros reutilizables que pueden quedar integradas en la interfaz de usuario.
- @personalizing-te3: Este artículo te guía por la amplia gama de opciones de configuración y preferencias disponibles en Tabular Editor 3 para ajustar la herramienta a tus necesidades.
