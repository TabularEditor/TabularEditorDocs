## Vienes de Tabular Editor 2.x?

Mira [este vídeo](https://www.youtube.com/watch?v=pt3DdcjfImY) para hacerte una idea de las nuevas funciones de Tabular Editor 3.

**Resumen de las principales características de Tabular Editor 3:**

- IDE totalmente personalizable, con compatibilidad con varios monitores, pantallas Hi-DPI y temas
- Nuevo y potente editor de código DAX con autocompletado, comprobación de sintaxis, plegado de código y mucho, mucho más
- \*Modo del área de trabajo, que te permite guardar tus cambios en el disco y sincronizar a la vez los metadatos del modelo con Analysis Services
- \*Previsualiza los datos de las tablas con desplazamiento infinito, crea PivotGrids o escribe consultas DAX para explorar el modelo o probar la lógica de los cálculos
- \*Actualiza los esquemas de tabla tanto en Data sources de proveedor como en orígenes de datos estructurados (sí, ¡incluso para consultas M!)
- \*Programa actualizaciones de datos
- Crea diagramas del Data model
- Crea scripts DAX que te permitan editar varias medidas u otros objetos calculados en un solo documento
- Graba C# Scripts y guárdalos como macros (antes conocidas como "Acciones personalizadas")
- Integración con el Analizador VertiPaq

\*=Solo mientras estés conectado a una instancia de Analysis Services o Power BI

## Problemas conocidos

- El Asistente de importación de tablas no se incluyó en esta versión. Estará disponible en la versión 3.0.3, que se publicará muy pronto. Como solución alternativa, simplemente crea una nueva tabla y especifica la consulta de partición (SQL o M). Después, haz clic con el botón derecho en la tabla y elige "Actualizar esquema de tabla..." para importar columnas.
- Consulta la [hoja de ruta](https://github.com/TabularEditor/TabularEditor3/issues/12) para obtener más información.

## Nuevas funciones en 3.0.1:

(La lista siguiente compara Tabular Editor 3.0.1 con la versión beta anterior)

- Se actualizó el [EULA](https://tabulareditor.com/tabular-editor-software-license-agreement_ver-1-0/) y se agregó [ThirdPartyNotices.txt](https://github.com/TabularEditor/TabularEditor3/blob/master/ThirdPartyNotices.txt) al instalador.
- Schema Compare ahora también funciona con Data Sources de Provider (Legacy) y con particiones de consulta
- La primera vez que uses el botón "Dar formato a DAX", Tabular Editor te pedirá tu consentimiento para enviar código DAX al servicio web www.daxformatter.com (una próxima versión de Tabular Editor 3 incluirá formato de código DAX sin conexión, por lo que este consentimiento dejará de ser necesario).
- Se mejoraron algunas cadenas de texto del Asistente de activación y ahora se puede introducir una nueva clave de licencia si la licencia actual ha caducado.
- Se agregó el editor de cadena de conexión a la cuadrícula de propiedades
- Se agregaron preferencias para ignorar globalmente ciertos tipos de diferencias de esquema en Herramientas > Preferencias > Comparación de esquemas.
