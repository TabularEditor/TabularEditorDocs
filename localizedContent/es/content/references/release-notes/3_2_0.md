## Mejoras en 3.2.0

- Nos enorgullece presentar el **Depurador de DAX** en esta actualización de Tabular Editor 3. Consulta [este artículo](https://docs.tabulareditor.com/te3/features/dax-debugger.html) para empezar.
- Se ha mejorado considerablemente el tiempo de inicio y el consumo de recursos de la aplicación.
- Hemos renovado el motor de personalización del diseño para abordar varios problemas (editores de expresiones que desaparecían, botones de la barra de herramientas que faltaban, etc.) **Ten en cuenta que cualquier personalización del diseño que hayas hecho anteriormente dejará de funcionar en la versión 3.2.0**.
- TOM actualizado a [19.32.0](https://www.nuget.org/packages/Microsoft.AnalysisServices.retail.amd64/).

## Correcciones de errores en 3.2.0

- Se corrigió un problema por el que el método `CallDaxFormatter` no respetaba el valor predeterminado de formato corto/largo; consulta [#324](https://github.com/TabularEditor/TabularEditor3/issues/324).
- Se corrigió un problema con el formato de las propiedades de un elemento de cálculo en un script DAX.
- Ahora se pueden cargar ensamblados de terceros desde ubicaciones distintas a la carpeta de instalación; consulta [#340](https://github.com/TabularEditor/TabularEditor3/issues/340).
- Se corrigió un problema con la corrección automática de fórmulas cuando se cambiaba el nombre de una columna a la que se hacía referencia en una llamada a las funciones CROSSFILTER o USERELATIONSHIP.

## Vienes de Tabular Editor 2.x?

Mira [este vídeo](https://www.youtube.com/watch?v=pt3DdcjfImY) para hacerte una idea de las nuevas funciones de Tabular Editor 3. Además, asegúrate de consultar nuestra [guía de inicio](https://docs.tabulareditor.com/onboarding/index.html).

**Descripción general de las funciones principales de Tabular Editor 3:**

- IDE totalmente personalizable, con compatibilidad con configuraciones multimonitor, pantallas HiDPI y temas
- Nuevo y potente editor de código DAX con autocompletado, comprobación de sintaxis, plegado de código y mucho, mucho más
- \*Modo del área de trabajo, que te permite guardar los cambios en disco y sincronizar simultáneamente los metadatos del modelo con Analysis Services
- \*Previsualiza datos de tablas con desplazamiento infinito, crea PivotGrids o escribe consultas DAX para explorar el modelo o probar la lógica de cálculo
- \*Programa actualizaciones de datos
- Actualiza los esquemas de las tablas tanto en el Data source del proveedor como en el origen de datos estructurado (sí, ¡incluso para consultas M!)
- Crea diagramas del Data model
- Crea scripts DAX que te permiten editar varias medidas u otros objetos calculados en un solo documento
- Graba C# Scripts y guárdalos como macros (antes conocidas como "Custom Actions")
- Integración con Analizador VertiPaq
- \*Depurador de DAX

\*=Solo cuando estés conectado a una instancia de Analysis Services o de Power BI
