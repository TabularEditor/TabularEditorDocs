## Mejoras en 3.1.4

- Se han añadido algunos accesos directos de la barra de herramientas a los menús (por ejemplo, ahora puedes administrar las reglas de BPA en **Tools > Manage BPA rules...**)
- Ahora puedes crear scripts de DAX para columnas calculadas y tablas calculadas seleccionándolas en el Explorador TOM y haciendo clic con el botón derecho
- Se ha mejorado el comportamiento de los mensajes de error en Pivot Grid
- Se ha añadido el botón "Iniciar Tabular Editor 3" al instalador; consulta [#202](https://github.com/TabularEditor/TabularEditor3/issues/202)

## Correcciones de errores en 3.1.4

- Las columnas y medidas con corchetes en sus nombres, así como las tablas con comillas simples en su nombre, ya no generan un error semántico falso; consulta [#239](https://github.com/TabularEditor/TabularEditor3/issues/239).
- Se corrigió un error que provocaba un error semántico falso al usar ciertas palabras como nombres de variables.
- Se corrigió un error que en algunos casos podía provocar un bloqueo al definir una variable en DAX.
- Se corrigió un error que provocaba un error semántico falso al usar TOPNSKIP con determinados argumentos.
- El método `SelectObject<T>` (y los métodos que lo encapsulan) ahora devuelve null cuando el usuario pulsa el botón "Cancel"; consulta [#247](https://github.com/TabularEditor/TabularEditor3/issues/247).
- Se corrigió un problema por el que los métodos `ExportProperties` / `ImportProperties` no funcionaban con colecciones de objetos; consulta [#245](https://github.com/TabularEditor/TabularEditor3/issues/245).
- Se corrigió un problema que impedía pegar elementos de cálculo; consulta [#237](https://github.com/TabularEditor/TabularEditor3/issues/237)
- Se corrigió un error de "Duplicate key" que se producía al actualizar el esquema de una tabla que contenía columnas con nombres similares (es decir, "Customer Key" y "CustomerKey").

## Vienes de Tabular Editor 2.x?

Mira [este vídeo](https://www.youtube.com/watch?v=pt3DdcjfImY) para hacerte una idea de las nuevas funciones de Tabular Editor 3.

**Descripción general de las principales características de Tabular Editor 3:**

- IDE totalmente personalizable, con compatibilidad con varios monitores, Hi-DPI y temas
- Nuevo y potente editor de código DAX con autocompletado, comprobación de sintaxis, plegado de código y mucho, mucho más
- \*Modo del área de trabajo, que te permite guardar los cambios en disco y sincronizar simultáneamente los metadatos del modelo con Analysis Services
- \*Previsualiza los datos de la tabla con desplazamiento infinito, crea PivotGrids o escribe consultas DAX para explorar el modelo o probar la lógica de cálculo
- \*Programa actualizaciones de datos
- Actualiza los esquemas de las tablas tanto en los Data sources de proveedor como en los orígenes de datos estructurados (sí, ¡incluso para consultas M!)
- Crea diagramas del Data model
- Crea scripts DAX que te permitan editar varias medidas u otros objetos calculados en un único documento
- Graba C# Scripts y guárdalos como macros (antes conocidas como "Custom Actions")
- Integración del Analizador VertiPaq

\*=Solo mientras estés conectado a una instancia de Analysis Services o Power BI
