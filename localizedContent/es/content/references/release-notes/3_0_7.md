## Problemas conocidos

- El Asistente para importar tablas no llegó a esta versión debido a algunos errores de alta prioridad que había que corregir. ¡Estará disponible lo antes posible! Como solución alternativa, crea una nueva tabla y especifica la consulta de la partición (SQL o M). Luego, haz clic con el botón derecho en la tabla y elige "Actualizar esquema de la tabla..." para importar columnas.
- Consulta la [hoja de ruta](https://github.com/TabularEditor/TabularEditor3/issues/12) para obtener más información.

## Mejoras en 3.0.7

- Se permite usar un compilador personalizado para C# Scripts, consulta la incidencia [#97](https://github.com/TabularEditor/TabularEditor3/issues/97)
- Ahora se pueden ocultar las advertencias de columnas de fecha/hora automáticas (en Herramientas > Preferencias > Tabular Editor > Ocultar advertencias de fecha/hora automática). Consulta la incidencia [#99](https://github.com/TabularEditor/TabularEditor3/issues/99)

## Correcciones de errores en 3.0.7

- "Standard Edition" ha pasado a llamarse "Edición Business"
- Se corrigió un problema del analizador semántico que, en ocasiones, podía provocar excepciones de desbordamiento de pila y el cierre de la herramienta (consulta la incidencia [#95](https://github.com/TabularEditor/TabularEditor3/issues/95)).
- Se corrigió un problema del analizador semántico que, en ocasiones, podía "Report" errores de dependencia circular inexistentes.
- El analizador semántico ahora admite la función DAX SUBSTITUTEWITHINDEX
- Se corrigió un problema con algunas de las funciones DAX JOIN (NATURALINNERJOIN, etc.) lo que en ocasiones devolvía un conjunto de columnas incorrecto (consulta el problema [#103](https://github.com/TabularEditor/TabularEditor3/issues/103))
- El autocompletado ahora solo muestra las columnas y tablas relevantes en RELATED y RELATEDTABLE (consulta el problema [#112](https://github.com/TabularEditor/TabularEditor3/issues/112))
- El autocompletado ahora funciona con ORDER BY (consulta el problema [#53](https://github.com/TabularEditor/TabularEditor3/issues/53))
- El editor ya no desaparece al crear una carpeta, copiar/pegar un único objeto, etc. (consulta el problema [#90](https://github.com/TabularEditor/TabularEditor3/issues/90))
- Ahora deberían mostrarse todos los elementos relevantes del menú contextual para los grupos de cálculo (incluidas las opciones de actualización), consulta el problema [#100](https://github.com/TabularEditor/TabularEditor3/issues/100)
- Se corrigió un problema por el que las consultas DAX no devolvían resultados al añadir una cláusula ORDER BY a la consulta, consulta el problema  [#108](https://github.com/TabularEditor/TabularEditor3/issues/108)
