Acción de código `RW003` (Reescrituras): **Invertir IF**

## Descripción

Para mejorar la legibilidad, a veces resulta útil invertir sentencias `IF`.

### Ejemplo

Cambiar:

```dax
IF(a < b, "B is greater", "A is greater")
```

A:

```dax
IF(a > b, "A is greater", "B is greater")
```

## ¿Por qué Tabular Editor sugiere esto?

Esta acción de código ofrece una forma rápida de invertir la lógica de una instrucción `IF`, lo que puede hacer que el código sea más fácil de leer. La legibilidad del código es subjetiva y depende del contexto; por eso, esta acción de código está en la categoría **Reescrituras** en lugar de la categoría **Legibilidad**.
