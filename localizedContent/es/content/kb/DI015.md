Acción de código `DI015` (Mejora) **Sustituir IF por DIVIDE**

## Descripción

Use la función [`DIVIDE`](https://dax.guide/DIVIDE) en lugar de [`IF`](https://dax.guide/IF) para comprobar más fácilmente si el denominador es cero o está en blanco.

### Ejemplo 1

Cambiar:

```dax
IF([Total Cost] = 0, BLANK(), [Total Sales] / [Total Cost])
```

Por:

```dax
DIVIDE([Total Sales], [Total Cost])
```

### Ejemplo 2

Cambiar:

```dax
IF([Total Cost] <> 0, [Total Sales] / [Total Cost])
```

Por:

```dax
DIVIDE([Total Sales], [Total Cost])
```

## ¿Por qué Tabular Editor sugiere esto?

La función `DIVIDE` es una forma más concisa y legible de gestionar la división entre cero o con valores en blanco. Al usar `DIVIDE`, haces que tu código sea más robusto y fácil de entender. Además, la función `DIVIDE` es más eficiente que usar `IF` para comprobar si el denominador es cero o está en blanco, ya que solo evalúa el denominador una vez.
