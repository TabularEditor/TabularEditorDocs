Acción de código `DI009` (Mejora) **Evitar la sintaxis abreviada de CALCULATE**

## Descripción

No use la sintaxis abreviada de calculate.

### Ejemplo

Cambia:

```dax
[Total Sales](Products[Color] = "Red")
```

Por:

```dax
CALCULATE([Total Sales], Products[Color] = "Red")
```

## ¿Por qué Tabular Editor sugiere esto?

La sintaxis abreviada de `CALCULATE` es un atajo de la función `CALCULATE`, donde el primer argumento es la medida que se va a evaluar y el segundo argumento es la expresión de filtro. Aunque esta sintaxis es válida, no se recomienda, ya que puede resultar confusa de leer y entender. Es mejor usar la función `CALCULATE` de forma explícita, ya que hace que el código sea más legible y fácil de mantener.
