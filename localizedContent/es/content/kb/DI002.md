Acción de código `DI002` (Mejora) **Eliminar todas las variables no utilizadas**

## Descripción

Las variables que no se usan (directa o indirectamente a través de otras variables) en la parte `RETURN` de un bloque de variables deben eliminarse.

## Ejemplo

Cambio:

```dax
VAR _internetSalesTaxed = [Internet Sales] * 1,25
VAR _cogs = [Cost of Sales]
VAR _cogsTaxed = _cogs * 1,25
RETURN _internetSalesTaxed
```

A:

```dax
VAR _internetSalesTaxed = [Internet Sales] * 1,25
RETURN _internetSalesTaxed
```

## ¿Por qué Tabular Editor sugiere esto?

Las variables no utilizadas pueden hacer que tu código sea más difícil de leer y entender. Al eliminarlas, haces que tu código sea más conciso y fácil de mantener.

Una variable no utilizada también es un indicio de que el código puede contener un error o de que la variable se pretendía usar para algo que más tarde se eliminó. Al eliminar la variable, evitas posibles confusiones para otros desarrolladores que puedan estar leyendo tu código.

Si una variable es referenciada por otras, pero ninguna de ellas se usa en la parte `RETURN` del bloque de variables, puedes eliminarlas todas sin problema.

## Observaciones

Esta acción de código tiene una variante **(Todas las apariciones)**, que se mostrará cuando se puedan mejorar varias secciones de código. Esta variante aplicará la acción de código a todas las secciones pertinentes del documento de una sola vez.
