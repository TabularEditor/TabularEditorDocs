Acción de código `RW001` (Reescrituras) **Reescribir TOTALxTD con CALCULATE**

## Descripción

Funciones como [`TOTALMTD`](https://dax.guide/TOTALMTD), [`TOTALQTD`](https://dax.guide/TOTALQTD) y [`TOTALYTD`](https://dax.guide/TOTALYTD) pueden reescribirse con la función [`CALCULATE`](https://dax.guide/CALCULATE).

### Ejemplo

De:

```dax
 TOTALYTD([Total Sales], 'Date'[Date])
```

A:

```dax
CALCULATE([Total Sales], DATESYTD('Date'[Date]))
```

## ¿Por qué Tabular Editor sugiere esto?

Esta reescritura es útil si quieres agregar filtros adicionales o modificar el contexto de cálculo.

> [!NOTE]
> Esta acción de código está en la categoría **Reescrituras**, lo que significa que no supone una recomendación general ni una buena práctica. En su lugar, esta acción de código ofrece una forma rápida de reescribir el código de forma distinta, por ejemplo, como parte de una refactorización más amplia. Después de aplicar la acción de código, y hasta que realices más ediciones, puede que veas una acción de código de **Mejora** o **Legibilidad** que sugiera revertir el código a su estado original.
