Acción de código `DR007` (Legibilidad) **Simplificar bloque de 1 variable**

## Descripción

Un bloque `VAR` con una sola variable se puede simplificar moviendo la expresión directamente a la parte `RETURN` del bloque. Esto supone que solo se hace referencia a la variable una vez y sin ningún modificador de contexto.

### Ejemplo

Cambiar:

```dax
VAR _sales = [Total Sales]
RETURN
    _sales * 1,25
```

Por:

```dax
[Total Sales] * 1,25
```

## ¿Por qué Tabular Editor sugiere esto?

Cuando las declaraciones `VAR` son lo bastante simples y la variable se usa exactamente una vez en la parte `RETURN` del código, sin ninguna modificación de contexto, no hace falta declararla. Esto hace que el código sea más conciso y fácil de leer.
