Acción de código `DR010` (Legibilidad) **Reescribir usando COALESCE**

## Descripción

En lugar de usar [`IF`](https://dax.guide/IF) para devolver el primer valor no en blanco de una lista de expresiones, use la función [`COALESCE`](https://dax.guide/COALESCE).

### Ejemplo

Cambie:

```dax
IF(
    ISBLANK(Product[Long Description]), 
    Product[Short Description], 
    Product[Long Description]
)
```

Por:

```dax
COALESCE(Product[Long Description], Product[Short Description])
```

## ¿Por qué Tabular Editor sugiere esto?

La función `COALESCE` es una forma más concisa y fácil de leer para devolver el primer valor no en blanco de una lista de expresiones. Al usar `COALESCE`, el código resulta más legible y la intención de la expresión queda más clara.
