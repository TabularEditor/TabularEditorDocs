Acción de código `DR002` (Legibilidad) **Usa un agregador en lugar de un iterador**

## Descripción

Usa una función agregadora en lugar de una función iteradora siempre que sea posible para simplificar el código.

### Ejemplo

Cambia:

```dax
SUMX(Sales, Sales[Line Amount])
```

Por:

```dax
SUM(Sales[Line Amount])
```

## ¿Por qué Tabular Editor sugiere esto?

Cuando solo necesitas agregar los valores de una sola columna, las funciones agregadoras ([`SUM`](https://dax.guide), [`MIN`](https://dax.guide), [`MAX`](https://dax.guide), etc.), usan una sintaxis más sencilla y concisa que sus funciones iteradoras equivalentes ([`SUMX`](https://dax.guide), [`MINX`](https://dax.guide), [`MAXX`](https://dax.guide), etc.) y deberían preferirse para que el código sea más legible.
