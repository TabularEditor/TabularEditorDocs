Acción de código `DI013` (Mejora) **Usa el operador de división en lugar de DIVIDE**

## Descripción

Cuando el segundo argumento de [`DIVIDE`](https://dax.guide/DIVIDE) es una constante distinta de cero, es más eficiente usar el operador de división.

### Ejemplo

Cambiar:

```dax
DIVIDE([Total Sales], 1.25)
```

A:

```dax
[Total Cost] / 1.25
```

## ¿Por qué Tabular Editor sugiere esto?

La función `DIVIDE` es una forma sólida de gestionar los errores de división entre cero, ya que devuelve un valor específico o (Blank) si el denominador es cero. Sin embargo, cuando se garantiza que el denominador no es cero —por ejemplo, al dividir entre una constante (distinta de cero)—, el operador de división `/` es más eficiente y más fácil de leer. Al usar el operador de división, el código queda más conciso y fácil de entender.
