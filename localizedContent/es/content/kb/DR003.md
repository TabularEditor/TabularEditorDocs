Acción de código `DR003` (Legibilidad) **Use VALUES en lugar de SUMMARIZE**

## Descripción

Cuando [`SUMMARIZE`](https://dax.guide/SUMMARIZE) solo especifica una única columna y esa columna pertenece a la tabla especificada en el primer argumento, el código puede escribirse de forma más concisa usando [`VALUES`](https://dax.guide/VALUES).

### Ejemplo

Cambie:

```dax
SUMMARIZE(Sales, Sales[Product Key])
```

A:

```dax
VALUES(Sales[Product Key])
```

## ¿Por qué Tabular Editor sugiere esto?

La función `SUMMARIZE` es una función potente que se puede usar para agrupar datos y calcular agregaciones. Sin embargo, cuando solo necesita obtener los valores distintos de una única columna, la función `VALUES` es más concisa y más fácil de leer. Al usar la función `VALUES`, el código se vuelve más legible y la intención de la expresión queda más clara.
