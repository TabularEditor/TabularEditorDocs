Acción de código `DI001` (Mejora) **Eliminar VAR no utilizado**

## Descripción

Las variables a las que no se hace referencia en ninguna parte deben eliminarse.

## Ejemplo

Cambia:

```dax
VAR _internetSalesTaxed = [Internet Sales] * 1.25
VAR _cogsTaxed = [Cost of Sales] * 1.25
VAR _internetSales = [Internet Sales]
RETURN _internetSalesTaxed - _cogsTaxed
```

Por:

```dax
VAR _internetSalesTaxed = [Internet Sales] * 1.25
VAR _cogsTaxed = [Cost of Sales] * 1.25
RETURN _internetSalesTaxed - _cogsTaxed
```

## ¿Por qué Tabular Editor sugiere esto?

Las variables sin usar pueden hacer que tu código sea más difícil de leer y entender. Al eliminarlas, haces que tu código sea más conciso y fácil de mantener.

Una variable sin usar también puede indicar que el código puede contener un error o que la variable se pretendía usar para algo que después se eliminó. Al eliminar la variable, evitas posibles confusiones para otros desarrolladores que puedan estar leyendo tu código.
