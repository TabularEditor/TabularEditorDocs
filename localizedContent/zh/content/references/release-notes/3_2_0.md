## 3.2.0 的改进

- 我们很高兴在这次 Tabular Editor 3 更新中推出 **DAX调试器**。 阅读[这篇文章](https://docs.tabulareditor.com/te3/features/dax-debugger.html)，即可开始使用。
- 应用启动时间和资源占用已大幅优化。
- 我们已全面重构布局自定义引擎，以解决多项问题（表达式编辑器消失、工具栏按钮缺失等） **请注意：你此前进行的任何布局自定义在 v. 3.2.0 中都将不再生效**。
- TOM 已更新至 [19.32.0](https://www.nuget.org/packages/Microsoft.AnalysisServices.retail.amd64/)。

## 3.2.0 的 Bug 修复

- 修复 `CallDaxFormatter` 方法未遵循短/长格式默认设置的问题，参见 [#324](https://github.com/TabularEditor/TabularEditor3/issues/324)。
- 修复 DAX 脚本中计算项属性格式化的问题。
- 现在可从安装文件夹以外的位置加载第三方程序集，参见 [#340](https://github.com/TabularEditor/TabularEditor3/issues/340)。
- 修复在重命名被 CROSSFILTER 或 USERELATIONSHIP 函数调用引用的列时，自动公式修复发生错误的问题。

## 从 Tabular Editor 2.x 升级过来？

看看[这个视频](https://www.youtube.com/watch?v=pt3DdcjfImY)，快速了解 Tabular Editor 3 的新功能。 另外，别忘了查看我们的[上手指南](https://docs.tabulareditor.com/onboarding/index.html)。

**Tabular Editor 3 主要功能概览：**

- 完全可自定义的 IDE，支持多显示器、Hi-DPI 以及主题
- 全新强大的 DAX 代码编辑器，支持自动补全、语法检查、代码折叠等诸多功能
- \*工作区模式，让你在将更改保存到磁盘的同时，同步将模型元数据同步到 Analysis Services
- \*支持无限滚动预览表数据，可创建 PivotGrid，或编写 DAX 查询来浏览模型或测试计算逻辑
- \*安排数据刷新
- 同时更新 Provider 数据源和 Structured数据源上的表架构（没错，连 M 查询也支持！）
- 创建 Data model 图表
- 创建 DAX 脚本，让你在同一份文档中编辑多个度量值或其他计算对象
- 录制 C# Script 并保存为宏（以前称为“Custom Actions”）
- 集成 VertiPaq分析器
- \*DAX调试器

\*=仅在连接到 Analysis Services 或 Power BI 实例时可用
