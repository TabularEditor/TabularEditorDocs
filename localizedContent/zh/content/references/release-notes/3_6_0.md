## 3.6.0 新增功能

查看我们的[发布博文](https://blog.tabulareditor.com/2023/04/19/tabular-editor-3-april-2023-release/)，快速了解此版本最重要的更新。

- Tabular Editor 3 现已原生支持 **Databricks**。 这意味着，你现在可以在使用“表导入向导”导入表时，连接到 Databricks SQL 端点。 想了解更多，可以看看我们的[发布博文](https://blog.tabulareditor.com/2023/04/19/tabular-editor-3-april-2023-release)。 这意味着，你现在可以在使用“表导入向导”导入表时，连接到 Databricks SQL 端点。 想了解更多，可以看看我们的[发布博文](https://blog.tabulareditor.com/2023/04/19/tabular-editor-3-april-2023-release)。
- 我们新增了对在 [Power BI Desktop 2023 年四月更新](https://powerbi.microsoft.com/en-us/blog/power-bi-april-2023-feature-summary/) 中引入的 DAX 新函数 [`RANK`](https://dax.guide/rank) 和 [`ROWNUMBER`](https://dax.guide/rownumber) 的支持。 此外，我们的代码补全现在会为窗口函数和 [`ORDERBY`](https://dax.guide/orderby) 函数的 &lt;Blanks&gt; 与 &lt;Order&gt; 参数显示所有可用关键字。 此外，我们的代码补全现在会为窗口函数和 [`ORDERBY`](https://dax.guide/orderby) 函数的 &lt;Blanks&gt; 与 &lt;Order&gt; 参数显示所有可用关键字。

## 3.6.0 中的小幅改进

- 在将更改保存到 Workspace 数据库之前，我们不再执行（可能较慢的）冲突检查。 我们假定每位开发者都使用自己的个人 Workspace 数据库，因此不会发生冲突。 我们假定每位开发者都使用自己的个人 Workspace 数据库，因此不会发生冲突。
- C# Script：`IAnnotationObject` 接口已扩展，新增 `ClearAnnotations()` 方法。
- 在为度量值添加动态格式字符串表达式时，我们现在会清除静态格式字符串属性（因为度量值无法同时设置静态和动态格式字符串）。
- 将 TOM 更新到 [19.61.1.4](https://www.nuget.org/packages/Microsoft.AnalysisServices.NetCore.retail.amd64)。
- 使用 Tabular Editor 创建的新 Power BI 模型的默认兼容级别现为 1601。 此外，“创建新模型”对话框现在允许你指定任意兼容级别。 此外，“创建新模型”对话框现在允许你指定任意兼容级别。

## 3.6.0 中的 Bug 修复

- 已更新 Oracle 下载说明，指导用户使用 ODAC（而非 ODP.NET）驱动程序。
- 在打开或新建数据库时，已移除对数据库名称中特殊字符的限制。
- 修复了一个问题：该问题会导致 .bim / Database.json 文件中的数据库名称属性被更新为 Workspace 数据库的名称。
- 提升了 DAX调试器的稳定性。
- 当未勾选“创建用户选项 (.tmuo) 文件”选项时，将不再保存“.tmuo”文件。
- 修复了代码补全有时会“吞掉”右括号的问题。
- 在度量值表达式和格式字符串表达式的上下文中使用 [`SELECTEDMEASURE`](https://dax.guide/selectedmeasure)（及相关函数）时，语义分析器不再报错。
- 修复了从现有隐式 (SQL) 数据源导入新表时生成的 M 查询在尝试刷新表时无法正常工作的问题。

观看[此视频](https://www.youtube.com/watch?v=pt3DdcjfImY)，快速了解 Tabular Editor 3 的新功能。 另外，别忘了查看我们的[入门指南](https://docs.tabulareditor.com/onboarding/index.html)。 另外，别忘了查看我们的[入门指南](https://docs.tabulareditor.com/onboarding/index.html)。

**Tabular Editor 3 主要功能概览：**

- 完全可自定义的 IDE，支持多显示器、Hi-DPI 和主题
- 全新强大的 DAX 代码编辑器，提供自动补全、语法检查、代码折叠等更多功能
- \*工作区模式：允许你在将更改保存到磁盘的同时，将模型元数据同步到 Analysis Services
- \*支持无限滚动预览表数据；可创建 PivotGrids 或编写 DAX 查询来浏览模型或测试计算逻辑
- \*计划定时刷新数据
- 在 Provider 数据源和 Structured数据源上更新表架构（没错，连 M 查询也支持！）
- 创建 Data model 示意图
- 创建 DAX 脚本，让你在同一文档中编辑多个度量值或其他计算对象
- 录制 C# Script 并保存为宏（以前称为“Custom Actions”）
- 集成 VertiPaq分析器
- DAX调试器

\*=仅在连接到 Analysis Services 或 Power BI 实例时才可用
