## 3.4.1 中的新功能

- 将 TOM 更新到 [19.55.3](https://www.nuget.org/packages/Microsoft.AnalysisServices.NetCore.retail.amd64)。
- 新增 [透视编辑器](xref:perspective-editor) 视图。

## 3.4.1 中的小幅改进

- DAX调试器现已支持窗口函数（[`OFFSET`](https://dax.guide/offset)、[`INDEX`](https://dax.guide/index) 和 [`WINDOW`](https://dax.guide/window)）。
- DAX调试器现在可调试在表构造器中引用的度量值，例如 `EVALUATE { [Internet Sales] }`，或在 [`ROW`](https://dax.guide/row) 函数中引用的度量值。
- **评估语境** 视图现在会显示用作筛选器的表变量表达式。
- 通过 **窗口** 菜单重置窗口布局后，将不再影响工具栏按钮（这些按钮可在 **工具 > 自定义** 对话框中修改或重置）。

## 3.4.1 的 Bug 修复

- 修复了偏好设置对话框中键盘命令列表上方的搜索框问题。
- DAX 编辑器现在可以正确处理嵌套注释，参见 [#681](https://github.com/TabularEditor/TabularEditor3/issues/681)。
- 尝试调试查询范围的度量值不再导致崩溃。
- 修复了图表视图中的关系连线会“卡住”的问题，参见 [#787](https://github.com/TabularEditor/TabularEditor3/issues/787)。
- 在依赖项视图中跟踪 TOM Explorer 的选中项时，选择关系也不会再导致应用程序崩溃，参见 [#782](https://github.com/TabularEditor/TabularEditor3/issues/782)。
- 修复了在尝试更改或重置分区的 DataSource 属性时出现的 “DataSourceConverter cannot convert from TabularEditor.TOMWrapper.ProviderDataSource” 错误。
- 使用“保存到文件夹”时，Tabular Editor 3 不会再将文件名末尾的空白字符替换为 `%20`。
- 表达式编辑器不再需要先点击标题才会显示内容，参见 [#739](https://github.com/TabularEditor/TabularEditor3/issues/739)。
- DAX调试器现在可正确模拟 [`SUMMARIZECOLUMNS`](https://dax.guide/summarizecolumns) 的 auto-exist 行为。
- 修复了一个问题：当使用 `ALL()`、`ALLSELECTED()` 等筛选移除函数时，**评估语境**视图中的外层筛选器不会被划删除线（表示这些筛选器已被覆盖）。
- 修复了右键菜单中缺少“刷新”选项、导致无法刷新计算表格或计算组的问题。
- 修复了在尝试执行架构更新时偶发导致崩溃的问题。
- 现在无法再调试已从模型中删除的度量值（否则会导致崩溃）。
- 在将对象粘贴到 TOM Explorer 时，改进了异常处理。
- 修复了一个问题：删除正在 **搜索结果**视图中显示的对象时会导致崩溃。
- 修复了一个问题：在 **表格预览**中删除列或表时会导致崩溃。
- 当窗口函数的 `<Relation>` 参数留空，或包含扩展列时，Semantic Analyzer 不再报告错误。 查看 [#807](https://github.com/TabularEditor/TabularEditor3/issues/807)。 查看 [#807](https://github.com/TabularEditor/TabularEditor3/issues/807)。
- “自定义”对话框中的“重置”按钮将不再从工具栏中移除所有按钮。
- 在 C# Script 编辑器中，按下这些按键之一触发自动完成时，不应再额外插入制表符或换行符。
- 修复了通过 C# 脚本调用 `DependsOn.Deep()` 方法时出现的问题。
- DAX 编辑器在尝试自动格式化代码时不应再崩溃。

**Tabular Editor 3 主要功能概览：**

- 完全可自定义的 IDE，支持多显示器、Hi-DPI 和主题
- 全新强大的 DAX 代码编辑器，支持自动完成、语法检查、代码折叠等更多功能
- \*工作区模式，可将更改保存到磁盘，并同时将模型元数据同步到 Analysis Services
- \*支持无限滚动预览表格数据，可创建 PivotGrids 或编写 DAX 查询来浏览模型或测试计算逻辑
- \*计划数据刷新
- 支持在 Provider 数据源和 Structured数据源上更新表架构（没错，连 M 查询也支持！）
- 创建数据模型图
- 创建 DAX 脚本，以便在单个文档中编辑多个度量值或其他计算对象
- 录制 C# Script 并将其保存为宏（原称“自定义操作”）
- 集成 VertiPaq分析器
- \*DAX调试器

\*=仅在连接到 Analysis Services 或 Power BI 实例时可用
