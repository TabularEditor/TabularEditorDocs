## 3.1.0 新增功能

- 表导入向导终于来了！ 而且比以往更好用。 更多信息请阅读[这篇文章](https://docs.tabulareditor.com/te3/importing-tables.html)。 目前支持 SQL、ODBC 和 OLE DB；如果你需要支持其他类型的数据源，请告诉我们。
- 现在在离线状态（即未连接到 Analysis Services）也能检测表架构更改——即使是 M 分区和隐式数据源也可以。
- 现在可以通过 TOM 的上下文菜单在透视中显示/隐藏对象，类似于 TE2。
- 现已提供“导入/导出翻译”选项，类似 TE2。
- 新增“批量重命名”对话框，类似 TE2（选择多个对象后按“F2”重命名时会弹出该对话框）。
- 新增“标记为日期表格”对话框，类似 TE2。

## 3.1.0 改进内容

- 从 DAX 查询结果中复制单个单元格时，现在会对值进行格式化，使其可以方便地粘贴到 DAX 表达式中。请参阅[#200](https://github.com/TabularEditor/TabularEditor3/issues/200)。
- 在关系图上下文菜单中新增“删除关系”选项。请参阅[#195](https://github.com/TabularEditor/TabularEditor3/issues/195)。
- 在 Pivot Grid 上下文菜单中新增“移除字段”选项。
- 创建表、刷新、导入表等模型级操作现已可通过“模型”菜单使用。
- Structured数据源的 AuthenticationKind 属性现在提供下拉列表。 Options 属性现在也可编辑，并且已公开 Query 属性。
- 已重新整理 TOM Explorer 的上下文菜单项，使其与 TE2 更加一致。
- 现在可通过右键单击角色直接编辑角色成员。
- 现在可从依赖关系视图将依赖项复制为文本/JSON 格式，类似于 TE2。
- “Basic”和“Bezier”皮肤新增调色板选项（它们基于矢量，因此在 Hi-DPI 下比类似 VS 的皮肤好看得多）。 在“Window”菜单底部可以找到新的调色板。

## 3.1.0 中的错误修复

- 修复了在对数据库进行排序时，“选择数据库”对话框出现的问题，参见 [#201](https://github.com/TabularEditor/TabularEditor3/issues/201)。
- 修复了“选择显示文件夹”对话框导致崩溃的问题，参见 [#199](https://github.com/TabularEditor/TabularEditor3/issues/199)。
- 修复了从 Classic Workspace 切换时发生崩溃的问题，参见 [#198](https://github.com/TabularEditor/TabularEditor3/issues/198)。
- 在表之间拖放计算列或度量值时，现在会保留翻译和透视信息，参见 [#194](https://github.com/TabularEditor/TabularEditor3/issues/194)。
- 修复了 M 代码中出现多余换行符的问题，参见 [#193](https://github.com/TabularEditor/TabularEditor3/issues/193)。
- 修复了“将表添加到关系图”对话框的焦点问题，参见 [#183](https://github.com/TabularEditor/TabularEditor3/issues/183)。
- 修复了在 Pivot Grid 的多个位置右键单击时发生崩溃的问题，参见 [#175](https://github.com/TabularEditor/TabularEditor3/issues/175)。
- 修复了外部变更跟踪干扰导致 TOM 被不必要地刷新的问题。
- 修复了工作区模式中关于“未保存的更改”的警告问题
- 部署角色时移除 memberId 属性，参见 [TE2 #906](https://github.com/TabularEditor/TabularEditor/issues/906)。
- 修复了 TOM Explorer 复制/粘贴操作的问题（可能与 [#184](https://github.com/TabularEditor/TabularEditor/issues/184) 有关）。

## 从 Tabular Editor 2.x 迁移过来？

观看 [此视频](https://www.youtube.com/watch?v=pt3DdcjfImY)，快速了解 Tabular Editor 3 的新功能。

**Tabular Editor 3 主要功能概览：**

- 可完全自定义的 IDE，支持多显示器、Hi-DPI 和主题
- 全新且强大的 DAX 代码编辑器，支持自动补全、语法检查、代码折叠等更多功能
- \*工作区模式：可让你将更改保存到磁盘，同时将模型元数据同步到 Analysis Services
- \*通过无限滚动预览表数据，创建 PivotGrid，或编写 DAX 查询来浏览模型或测试计算逻辑
- \*安排数据刷新
- 同时更新 Provider 数据源和 Structured数据源上的表架构（是的，甚至包括 M 查询！）
- 创建 Data model 示意图
- 创建 DAX 脚本，让你在一个文档中批量编辑多个度量值或其他计算对象
- 录制 C# Script 并保存为宏（以前称为“自定义操作”）
- VertiPaq分析器集成

\*=仅在连接到 Analysis Services 或 Power BI 实例时可用
