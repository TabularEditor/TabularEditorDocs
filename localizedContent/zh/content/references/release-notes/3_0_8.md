## 3.0.8 中的改进

- 新增选项：在 DAX 查询中仅执行选中的 EVALUATE 语句（快捷键 Shift+F5），参见议题 [#47](https://github.com/TabularEditor/TabularEditor3/issues/47)。
- 支持使用快捷键 Ctrl+/ 注释/取消注释代码，参见议题 [#46](https://github.com/TabularEditor/TabularEditor3/issues/46)。 你也可以在“工具 > 偏好 > DAX编辑器 > 常规”里更改注释样式（-- 或 //）。
- DAX脚本现在也会为当前所选内容中的 KPI 度量值、计算列和计算表格生成脚本。 我们还会添加注释，让代码更易读（可在“工具 > 偏好 > DAX脚本编写”里禁用）。
- 新增选项：当连接到 Power BI 模型时，可在所有 DAX 表达式开头自动添加一个换行符。 这样在 Power BI Desktop 的公式栏中，代码看起来更美观。 可在“工具 > 偏好 >”中禁用，参见议题 [#26](https://github.com/TabularEditor/TabularEditor3/issues/26)。
- 添加 Windows AD 角色成员时，现在会显示“目录对象选择器”对话框，参见议题 [#152](https://github.com/TabularEditor/TabularEditor3/issues/152)。
- “添加部署元数据”选项现已可在“工具 > 偏好 > 模型部署”下找到。
- Pivot Grid 改进：
  - 你现在可以将 TOM Explorer 对象拖放到 Pivot Grid 的所有区域中，详见 [#49](https://github.com/TabularEditor/TabularEditor3/issues/49)。
  - 已在 TOM Explorer 的上下文菜单中新增选项，可将度量值、列和层级直接添加到 Pivot Grid。
  - 在 Pivot Grid 中新增“显示字段”按钮，用于显示连接的数据库中所有可用字段。
  - 在 Pivot Grid 中新增右键菜单选项“转到对象”。
- 关系错误（路径不明确、状态无效等） 现已可在 TOM Explorer 中检测并标记

## 3.0.8 中的错误修复

- 将“执行查询”感叹号的颜色从红色改为绿色。
- 修复了在图表视图中反转关系时发生的崩溃
- 修复了关系编辑器中更改无法正确保存的问题
- 修复了当本机正在运行 SSAS 实例时，从 Power BI Desktop 启动 TE3 会发生的崩溃。
- 修复了在 Power BI XMLA endpoint 上使用 Workspace 数据库时无法创建新模型的错误。
- 修复了使用“注释”操作后代码选择会被清除的错误。
- Deployment Wizard 现在会记住上次使用的服务器。
- DAX脚本中的计算列表达式不再会因缺少行语境而误报语义错误信息。
- 修复了一个导致商业版许可证持有者无法使用 Visual Studio 集成 Workspace 服务器的错误。
- 修复了“successful”的拼写错误。
- 修复了自动格式化程序在输入右括号时有时会重复代码的问题。
- 在 UI 中新增“保存 Model.bim 备份”选项，参见问题 [#37](https://github.com/TabularEditor/TabularEditor3/issues/37)。
- “透视”选择不再会卡住，参见问题 [#52](https://github.com/TabularEditor/TabularEditor3/issues/52)。
- 属性网格现在能正确显示日期格式示例，参见问题 [#86](https://github.com/TabularEditor/TabularEditor3/issues/86)。
- 修复了搜索时的高亮颜色问题，参见问题 [#98](https://github.com/TabularEditor/TabularEditor3/issues/98)。
- “全部查找”现已按预期工作，参见问题 [#107](https://github.com/TabularEditor/TabularEditor3/issues/107)。
- 会导致度量值数据类型降级为 Variant 的计算项现已能被正确识别，见问题 [#111](https://github.com/TabularEditor/TabularEditor3/issues/111)。
- 修复在 FILTER 中使用 EARLIER 时误报语义错误的问题，见问题 [#118](https://github.com/TabularEditor/TabularEditor3/issues/118)。
- 修复在 FILTER 表达式中使用 KEEPFILTERS 时误报语义错误的问题，见问题 [#122](https://github.com/TabularEditor/TabularEditor3/issues/122)。
- 空白（或仅包含注释）的“详细行表达式”不再触发错误提示，见问题 [#127](https://github.com/TabularEditor/TabularEditor3/issues/127)。
- 修复重复弹出凭据提示的问题，见问题 [#131](https://github.com/TabularEditor/TabularEditor3/issues/131)。
- 修复 vpax 导入问题，见问题 [#133](https://github.com/TabularEditor/TabularEditor3/issues/133)。
- 修复选择显示文件夹时属性网格的表现，见问题 [#134](https://github.com/TabularEditor/TabularEditor3/issues/134)。
- 在图示中按下“Delete”不再删除表（改为从图示中移除），见问题 [#139 ](https://github.com/TabularEditor/TabularEditor3/issues/139)。
- Pivot Grid 现在可以正确处理日期字段，见问题 [#142](https://github.com/TabularEditor/TabularEditor3/issues/142)。
- 复制计算组时不再创建名称重复的度量值，同时也会调整优先级编号以确保唯一性，见问题 [#155](https://github.com/TabularEditor/TabularEditor3/issues/155) 和 [#156](https://github.com/TabularEditor/TabularEditor3/issues/156)。
- “取消注释代码”按钮现在也适用于带缩进的行，见 [#159](https://github.com/TabularEditor/TabularEditor3/issues/159)。
