# 键盘快捷键

自 2.7.3 版本起，Tabular Editor 支持以下键盘快捷键：

| 常规                                                     | 快捷键                                  |
| ------------------------------------------------------ | ------------------------------------ |
| 新建模型                                                   | Ctrl+N                               |
| 从文件加载模型                                                | Ctrl+O                               |
| 从数据库加载模型                                               | Ctrl+Shift+O                         |
| 保存模型                                                   | Ctrl+S                               |
| 复制所选内容                                                 | Ctrl+C                               |
| 剪切所选内容                                                 | Ctrl+X                               |
| 粘贴                                                     | Ctrl+V                               |
| 撤销                                                     | Ctrl+Z                               |
| 重做                                                     | Ctrl+Y                               |
| 全选                                                     | Ctrl+A                               |
| ~启动 Deployment Wizard~ | ~F6~ |
| 启动 Best Practice Analyzer                              | F10                                  |

| 表达式编辑器      | 快捷键          |
| ----------- | ------------ |
| 查找          | Ctrl+F       |
| 查找和替换       | Ctrl+H       |
| 转到定义        | F12          |
| 后退          | Alt+左箭头      |
| 前进          | Alt+右箭头      |
| 格式化 DAX     | F6           |
| 格式化 DAX（短行） | Ctrl+F6      |
| 注释行         | Ctrl+Shift+C |
| 取消注释        | Ctrl+Shift+U |

| 脚本编辑器 | 快捷键    |
| ----- | ------ |
| 查找    | Ctrl+F |
| 查找和替换 | Ctrl+H |
| 运行脚本  | F5     |

| 资源管理器树状视图       | 快捷键              |
| --------------- | ---------------- |
| 向上或向下移动         | 上/下箭头            |
| 展开/折叠当前节点       | 右/左箭头            |
| 展开/折叠当前节点及所有子节点 | Ctrl+右/左箭头       |
| 展开/折叠整个树形结构     | Ctrl+Shift+右/左箭头 |
| 切换度量值           | Ctrl+1           |
| 切换列             | Ctrl+2           |
| 切换层次结构          | Ctrl+3           |
| 切换显示文件夹         | Ctrl+4           |
| 切换显示隐藏对象        | Ctrl+5           |
| 切换元数据列          | Ctrl+F1          |
| 切换按字母顺序排序       | Ctrl+F2          |
| 切换所有对象类型        | Ctrl+F3          |
| 筛选              | Ctrl+F           |
| 切换筛选            | Ctrl+Shift+F     |
| 编辑表达式           | Enter            |
| 后退              | Alt+左箭头          |
| 前进              | Alt+右箭头          |
| 显示依赖项           | F3               |
| 设为不可见           | Ctrl+I           |
| 设为可见            | Ctrl+U           |
| 新建度量值           | Alt+1            |
| 新建计算列           | Alt+2            |
| 新建层次结构          | Alt+3            |
| 新建数据列           | Alt+4            |
| 新建表             | Alt+5            |
