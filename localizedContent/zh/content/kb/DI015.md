代码操作 `DI015`（改进）：**用 DIVIDE 替换 IF**

## 描述

使用 [`DIVIDE`](https://dax.guide/DIVIDE) 函数替代 [`IF`](https://dax.guide/IF)，更方便检查分母是否为零或空白。

### 示例 1

修改前：

```dax
IF([Total Cost] = 0, BLANK(), [Total Sales] / [Total Cost])
```

修改后：

```dax
DIVIDE([Total Sales], [Total Cost])
```

### 示例 2

修改前：

```dax
IF([Total Cost] <> 0, [Total Sales] / [Total Cost])
```

修改后：

```dax
DIVIDE([Total Sales], [Total Cost])
```

## 为什么 Tabular Editor 会建议这样做？

`DIVIDE` 函数能以更简洁、更易读的方式处理除零或空白值时的除法。 使用 `DIVIDE` 可使代码更健壮，也更易理解。 此外，`DIVIDE` 比用 `IF` 检查除数为零或空白值更高效，因为它只会对分母求值一次。 使用 `DIVIDE` 可使代码更健壮，也更易理解。 此外，`DIVIDE` 比用 `IF` 检查除数为零或空白值更高效，因为它只会对分母求值一次。
