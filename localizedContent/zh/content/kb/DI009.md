代码操作 `DI009`（改进）**避免使用 CALCULATE 快捷语法**

## 说明

不要使用 calculate 快捷语法。

### 示例

将以下内容：

```dax
[Total Sales](Products[Color] = "Red")
```

改为：

```dax
CALCULATE([Total Sales], Products[Color] = "Red")
```

## 为什么 Tabular Editor 会建议这样做？

CALCULATE 快捷语法是 `CALCULATE` 函数的简写：第一个参数是要进行求值的度量值，第二个参数是筛选表达式。 虽然这种语法是有效的，但不建议使用，因为它可能会让人难以阅读和理解。 更好的做法是显式使用 `CALCULATE` 函数，这样代码更易读，也更便于维护。
