代码操作 `DI001`（改进）**删除未使用的 VAR 变量**

## 说明

未在任何地方被引用的变量应当删除。

## 示例

修改前：

```dax
VAR _internetSalesTaxed = [Internet Sales] * 1.25
VAR _cogsTaxed = [Cost of Sales] * 1.25
VAR _internetSales = [Internet Sales]
RETURN _internetSalesTaxed - _cogsTaxed
```

修改为：

```dax
VAR _internetSalesTaxed = [Internet Sales] * 1.25
VAR _cogsTaxed = [Cost of Sales] * 1.25
RETURN _internetSalesTaxed - _cogsTaxed
```

## Tabular Editor 为什么会提出此建议？

未使用的变量会使代码更难阅读和理解。 删除它们后，代码会更精简，也更易于维护。

未使用的变量也表明代码可能包含错误，或者该变量原本打算用于某些用途，但后来相关内容被删除了。 通过删除该变量，可以避免其他可能阅读你代码的开发者产生困惑。
