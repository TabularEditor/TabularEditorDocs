代码操作 `DR003`（可读性）**使用 VALUES 替代 SUMMARIZE**

## 描述

当 [`SUMMARIZE`](https://dax.guide/SUMMARIZE) 仅指定单个列，且该列属于第一个参数所指定的表时，可以使用 [`VALUES`](https://dax.guide/VALUES) 更简洁地编写代码。

### 示例

将以下内容：

```dax
SUMMARIZE(Sales, Sales[Product Key])
```

替换为：

```dax
VALUES(Sales[Product Key])
```

## Tabular Editor 为什么会建议这样做？

`SUMMARIZE` 函数功能强大，可用于对数据分组并计算聚合值。 但当你只需要获取一列的不同值时，`VALUES` 函数更简洁，也更易读。 使用 `VALUES` 函数后，代码可读性更好，表达式的意图也更清晰。
