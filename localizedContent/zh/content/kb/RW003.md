代码操作 `RW003`（重写）：**反转 IF**

## 描述

为了提高可读性，有时把 `IF` 语句的逻辑反过来写会更清晰。

### 示例

改为：

```dax
IF(a < b, "B is greater", "A is greater")
```

改为：

```dax
IF(a > b, "A is greater", "B is greater")
```

## 为什么 Tabular Editor 会建议这样做？

这个代码操作提供了一种快速方式来反转 `IF` 语句的逻辑，从而让代码更易读。 代码的可读性具有主观性，也取决于具体上下文。因此，此代码操作归入 **重写** 类别，而非 **可读性** 类别。 代码的可读性具有主观性，也取决于具体上下文。因此，此代码操作归入 **重写** 类别，而非 **可读性** 类别。
