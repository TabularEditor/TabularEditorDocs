本节包含多篇文章，帮助你快速熟悉 Tabular Editor 3 及其主要功能，并重点介绍如何以尽可能高效的方式完成工作。

下面是文章概览以及你将在每篇文章中学到的内容：

- @importing-tables-data-modeling: 本文介绍如何导入表、更新表架构，以及如何可视化并编辑关系。
- @refresh-preview-query: 本文介绍如何安排刷新操作、预览表数据、使用 Pivot Grid 查看汇总数据、创建并执行 DAX 查询，以及如何使用 Tabular Editor 3 集成的 [VertiPaq分析器](https://www.sqlbi.com/tools/vertipaq-analyzer/)。
- @creating-and-testing-dax: 本文介绍 Tabular Editor 3 强大的 DAX 编辑器，并演示如何快速向模型添加和编辑 DAX 计算，例如度量值、计算列和计算表格。
- @dax-script-introduction: 本文演示如何使用 DAX 脚本功能，通过将多个度量值的定义合并到一个 DAX 脚本中，来编写、维护并测试跨多个度量值的复杂业务逻辑。
- @bpa: 本文提供有关 Best Practice Analyzer 的更多信息，并介绍如何设置可自定义的最佳实践规则，以提升模型的代码质量。
- @cs-scripts-and-macros: 本文介绍 Tabular Editor 3 中 C# Script 的概念、Script recorder，以及如何将脚本保存为可复用的宏，并将其集成到 UI 中。
- @personalizing-te3: 本文将带你了解 Tabular Editor 3 提供的丰富配置选项和偏好设置，帮助你按需调整工具。
