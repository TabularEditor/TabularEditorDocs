## 导入/导出翻译

在资源管理器树中选择一个或多个区域设置，右键单击并选择“导出翻译...”，即可生成一个 .json 文件，之后可在 Tabular Editor 或 Visual Studio 中导入。 选择“导入翻译...”以导入相应的 .json 文件。 可以选择是否覆盖现有翻译。 如果不选择覆盖，.json 文件中定义的翻译只会应用于该区域设置下尚未有翻译的对象。
