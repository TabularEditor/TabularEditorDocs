## 加载/保存 Model.bim 文件

按下 CTRL+O 会打开“打开文件”对话框，你可以在其中选择要在 Tabular Editor 中加载的 Model.bim 文件。 该文件的兼容级别必须为 1200 或更高（JSON 格式）。 CTRL+S 会将你在 Tabular Editor 中所做的更改保存回文件（建议在使用 Tabular Editor 之前先备份 Model.bim 文件）。 如果要将已加载的模型部署到 Analysis Services 服务器实例，请参阅 [部署](../features/deployment.md)。 该文件的兼容级别必须为 1200 或更高（JSON 格式）。 CTRL+S 会将你在 Tabular Editor 中所做的更改保存回文件（建议在使用 Tabular Editor 之前先备份 Model.bim 文件）。 如果要将已加载的模型部署到 Analysis Services 服务器实例，请参阅 [部署](../features/deployment.md)。
